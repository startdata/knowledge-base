이 대화에서 다룬 개념·키워드를 `~/knowledge-base`의 정리 플로우에 따라 정리한다. (그 폴더의 `CLAUDE.md`를 먼저 읽고 규칙을 따른다.)

절차:
1. 이 대화에서 **깊이 다룬** 개념·키워드를 선별한다 (무관한 잡담·단순 확인·실패한 시도는 제외).
2. 넘길 후보를 사용자에게 먼저 보여주고 확인받는다 — "이걸 정리해서 넘길게요: [[키워드A]](용도), [[키워드B]](스쳐감→키워드만). 뺄 것/더할 것 있나요?"
3. 확인 후 저장한다:
   - 보편 지식 → `~/knowledge-base/개념/<키워드>.md` 에 누적(없으면 생성, 같은 개념을 새 파일로 만들지 않는다)
   - 프로젝트 맥락·시행착오·수치·검증 → `~/knowledge-base/세션/<주제>/` 에 (프로젝트 기록)
   - 관련 키워드는 `[[링크]]`로 연결
4. `~/knowledge-base`에서 `git pull` → `git add . && git commit && git push` (커밋 메시지는 한국어).

> 이 파일은 백업본이다. `~/.claude/commands/마무리.md`로 복사해야 `/마무리` 커맨드로 동작한다.
