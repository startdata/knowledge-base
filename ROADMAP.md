# ROADMAP — 나중에 구현할 것

## 복습 봇 (Discord)
- 복습할 개념 질문을 Discord 채팅창에 자동 생성
- 내가 답변하면 → 기록(개념/세션 노트)과 대조 → 정답률 + 잘못 알고 있는 부분 피드백
- 소스: `개념/`·`세션/` 노트의 핵심 + 복습 카드(`#flashcards`)
- 상태: 아이디어. knowledge-base에 정리가 충분히 쌓인 뒤 착수.

## 정리 자동화 트리거
- 1차: commit 시점(Claude가 commit 실행 시 감지) → "정리할까요?"
- 2차: 세션 종료(`/마무리` 커맨드)
- 3차: 오래 정리 안 함(마지막 정리 후 일정 경과)
- 정리 담당 모델: Haiku (비용 최소)
- 흐름: 트랜스크립트 선별 → "넘길 후보 확인" → 저장·push
- 처음엔 1차만, 놓치면 2·3차 추가 (과설계 방지)
